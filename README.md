# Virtual Tour Creator 🏞️

Приложение для создания виртуальных 360° туров из фотографий Insta360 X4.

## Возможности

- 📁 Загрузка папок с панорамами
- 🗺️ Создание точек перехода между панорамами
- 🎮 Режим просмотра с навигацией
- 💾 Сохранение и загрузка проектов
- 🚀 Экспорт в HTML

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run dist
```

## Использование

1. Загрузите папку с панорамами
2. Добавьте точки перехода
3. Сохраните проект
4. Экспортируйте тур
